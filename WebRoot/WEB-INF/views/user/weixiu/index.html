维修须知

本楼盘内所有小区都设维修科。在主任的领导下，为业主提供房屋装修房屋，设施的维修及养护，并经常对楼房、设施设备进行巡视，做好巡检记录，及时满足业主维修要求，竭诚为您服务。请业主们就在以下条款范围内出现维修问题等情况及时进行报修。

一、 公用设施范围

房屋本体外墙面、屋面、楼梯通道外墙面

小区内道路面、人行道

小区内照明楼道灯、广场灯、车库灯、草坪灯、

沟渠池井雨水口、雨水井、污水井

停车场地面、标识牌、消防设施

地下停车场、电梯、公共设施围栏、宣传栏、广播系统

二、业主报修范围

房屋维修：屋面、外墙、房屋承重结构、走廊、通道、楼梯

室外照明、室外卫生、屋内管道、屋内排水

屋内供水、供电、煤气，有线电视、通信

三、报修流程：

登录楼盘物业管理系统 点击“维修管理” 阅读“维修须知”

点击“在线维修”，填写详情，提交

我们承诺接到报修后，3分钟内落实处理措施，在30分钟内落实解决方案，上门维修，维修完工后及时回访。
